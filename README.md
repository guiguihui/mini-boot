# mini-boot

#### 介绍
停车管理系统后台

#### 软件架构
用了Spring Boot、MyBaits-puls、MySQL


#### 安装教程
放进idea就可以用，记得要改一下字符格式，改成utf-8

#### 使用说明
test里边有个Test类，这个里边放的是车牌识别结果，格式如下：
桂AE0886 2022-05-19 14:13:55
